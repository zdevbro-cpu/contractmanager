# 4대보험 급여 계산 정책

## 적용 배경

현재 시스템은 사업소득 방식으로 3.3% 원천징수 후 지급.
4대보험 가입을 희망하는 임직원은 별도 계산 방식 적용.

---

## 사업소득 3.3% (기본)

```
지급액 = (연봉/12 + 활동비) × 96.7%
```

- 계약자가 프리랜서/개인사업자 신분
- 회사 추가 부담 없음

---

## 4대보험 가입 시 (2025년 기준)

### 근로자 공제 요율

| 항목 | 요율 |
|------|------|
| 국민연금 | 4.5% |
| 건강보험 | 3.545% |
| 장기요양보험 | 건강보험료 × 12.95% |
| 고용보험 | 0.9% |
| 소득세 | 간이세액표 (부양가족 1인 고정) |
| 지방소득세 | 소득세 × 10% |
| **합계** | **약 9.4% + 소득세** |

### 사업주 추가 부담

| 항목 | 요율 |
|------|------|
| 국민연금 | 4.5% |
| 건강보험 | 3.545% |
| 고용보험 | 0.9%~1.65% |
| 산재보험 | 업종별 상이 |
| **합계** | **약 10~11%** |

## 활동비 처리 방침

활동비가 50만원 이상인 경우:

| 항목 | 금액 | 처리 |
|------|------|------|
| 식대 | 20만원 | 비과세 |
| 자가운전보조금 | 20만원 | 비과세 |
| **비과세 한도** | **40만원** | 4대보험 기준 제외 |
| 초과분 | 활동비 - 40만원 | 과세 수당 (4대보험 기준 포함) |

### 계산 공식

```
세전 월급     = 연봉 / 12
비과세 활동비 = min(활동비, 400,000)
과세 수당     = max(활동비 - 400,000, 0)
보험료 기준   = 세전월급 + 과세수당

국민연금      = 보험료기준 × 4.5%
건강보험      = 보험료기준 × 3.545%
장기요양      = 건강보험료 × 12.95%
고용보험      = 보험료기준 × 0.9%
소득세        = 간이세액표(보험료기준, 부양가족 1인)
지방소득세    = 소득세 × 10%

실수령 = (세전월급 + 활동비) - 공제 합계
```

## 시스템 적용 방침

- 계약자별 보험 유형: `사업소득` / `4대보험` 선택
- 부양가족수: 1인 고정 (연말정산 시 환급/추가납부)
- 활동비 비과세 한도: 40만원 고정 적용
- 국민연금 상한 월보수: 590만원 (2025년 기준)
